# ScaleNPU

Welcome to the ScaleNPU repository! This repository contains the hardware and software components for the ScaleNPU module.

## Overview

The ScaleNPU is an open-source AI hardware accelerator specifically designed for neural network inference tasks. It forms part of the H-SCALE project.

## Official Documentation

For detailed information on the ScaleNPU's architecture details, please refer to the official documentation:

[ScaleNPU Documentation](https://opencehardware.github.io/ScaleNPU/)

## Getting Started

To get started with the ScaleNPU, clone this repository and navigate to the relevant directory:

```bash
git clone https://github.com/OpenCEHardware/ScaleNPU
cd ScaleNPU
```

Follow the instructions in the official documentation to configure and utilize the NPU effectively.
